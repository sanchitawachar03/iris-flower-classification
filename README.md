# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project predicts the species of an Iris flower using Machine Learning classification algorithms. The model is trained on the Iris dataset and evaluated using test data.

The project demonstrates the complete Machine Learning workflow, including data loading, train-test split, model training, prediction, and accuracy evaluation.

## ✨ Features

- Load Iris dataset from Excel files
- Train Machine Learning models
- Predict Iris flower species
- Compare Decision Tree and Random Forest classifiers
- Evaluate model accuracy

## 🛠️ Technologies Used

- Python
- NumPy
- Scikit-learn
- Jupyter Notebook
